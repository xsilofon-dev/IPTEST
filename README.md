# IPTEST
Інструмент для моніторингу доступності IP-адрес та мережевих діапазонів.

## Основний функціонал
- `check_ips.py`: Головний скрипт для перевірки IP (порт 443).
- `ips.txt`: Список цілей (IP та CIDR).
- `port_scanner.py`: Утиліта для швидкого сканування портів.

## Історія створення (Пам'ять сесії)
Проект створено з метою автоматизації моніторингу мережі. Основні етапи:
1. Створення логіки перевірки через сокети (обхід обмежень `ping`).
2. Додавання кольорової візуалізації (ANSI).
3. Впровадження підтримки CIDR-діапазонів.
4. Налаштування безпечного SSH-доступу до GitHub.
5. Публікація репозиторію через GitHub CLI (`gh`).

## Налаштування для SSH
Для роботи з репозиторієм використано ключ, згенерований у файлі `~/xsilofon`.
Конфігурація SSH (`~/.ssh/config`):
```text
Host github.com
  HostName github.com
  User git
  IdentityFile ~/xsilofon
```

## Автори
Проект розроблено в ході інтерактивної сесії Gemini CLI для особистого моніторингу мережі.
